# Hope Agent 0.15.0 — 体验细节打磨

> **简体中文** · [English](https://github.com/shiwenwen/hope-agent/blob/v0.15.0/docs/release-notes/v0.15.0.en.md)

> 发布日期：2026-06-30

> 详见 [`CHANGELOG.md`](https://github.com/shiwenwen/hope-agent/blob/v0.15.0/CHANGELOG.md#0150---2026-06-30)。

v0.15.0 是一次小幅体验优化：统一了侧边栏右键菜单的浮层风格，并理清了「快捷输入」与「快捷指令」两套入口的术语，让输入框的提示更准确。

## 改进

- **对话列表右键菜单更贴近输入框浮层风格**：侧边栏会话、项目、Agent 与分组筛选的右键菜单改用与对话输入框模型选择器一致的浮层样式，视觉更统一；项目右键菜单里的「新建会话」文案也更短，避免菜单过宽。
- **快捷输入与快捷指令术语区分更清晰**：输入框 `#` 唤起的小抄统一改称「快捷输入」，与 `/` 唤起的「快捷指令」（斜杠命令）区分开，相关菜单与提示文案 12 种界面语言同步更新；输入框 placeholder 也补充了 `/` 快捷指令的提示。
