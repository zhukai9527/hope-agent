# Hope Agent 0.15.0 — Experience Polish

> [简体中文](https://github.com/shiwenwen/hope-agent/blob/v0.15.0/docs/release-notes/v0.15.0.md) · **English**

> Released: 2026-06-30

> See [`CHANGELOG.md`](https://github.com/shiwenwen/hope-agent/blob/v0.15.0/CHANGELOG.md#0150---2026-06-30) for the full list.

v0.15.0 is a small experience pass: it unifies the floating style of the sidebar context menus and clarifies the terminology between "quick inputs" and "quick commands", so the input box hints read more accurately.

## Improvements

- **Sidebar context menus match the input-box overlay style**: the right-click menus for sidebar sessions, projects, agents, and group filters now use the same floating style as the model selector in the conversation input box for a more consistent look; the "New session" label in the project context menu is also shortened to keep the menu from getting too wide.
- **Clearer terminology between quick inputs and quick commands**: the cheat sheet summoned by `#` in the input box is now consistently called "quick inputs", distinct from the "quick commands" (slash commands) summoned by `/`; the related menu and hint copy are updated across all 12 UI languages, and the input-box placeholder now also hints at `/` quick commands.
