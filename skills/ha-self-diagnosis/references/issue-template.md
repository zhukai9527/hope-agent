# Issue Templates

Use these shapes when building the `body` for `issue_report`.

## Bug

```markdown
## Summary

<one-paragraph symptom>

## Environment

- Hope Agent version:
- Run mode:
- Platform:
- Relevant provider/channel/tool:

## Steps to Reproduce

1.

3.

## Expected Behavior

<what should have happened>

## Actual Behavior

<what happened instead>

## Diagnostic Evidence

<redacted logs, session/tool rows, error snippets, and related source/doc references>

## Possible Cause

<best current hypothesis, or "unknown">
```
